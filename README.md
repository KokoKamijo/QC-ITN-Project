## 📌 Project Overview

QC-ITN is a speculative network architecture designed for real‑time traffic coordination across planets. It combines:

- **Quantum Entanglement** for zero‑latency synchronization (simulated)
- **Mycelium‑inspired computing** for adaptive local pathfinding
- **DNA storage** for high‑density traffic logging
- **Neuromorphic AI** for predictive transport protocols

The project simulates traffic between Earth, Moon, and Mars using Python, demonstrating how next‑generation networks could overcome interplanetary latency.
